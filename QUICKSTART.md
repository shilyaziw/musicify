# Musicify 快速开始指南 🚀

欢迎使用 Musicify! 这份5分钟指南将帮助你快速上手。

---

## 📦 安装

### 全局安装 (推荐)

```bash
npm install -g ai-musicify
```

### 本地开发

```bash
git clone https://github.com/wordflowlab/musicify.git
cd musicify
npm install
npm run build
```

---

## ✨ 创建第一首歌

### 第1步: 初始化项目

```bash
musicify init "我的第一首歌"
cd "我的第一首歌"
```

你会看到交互式选择界面:
1. 选择 AI 助手 (推荐 Claude 或 Cursor)
2. 选择歌曲类型 (如: 流行)
3. 选择脚本类型 (macOS/Linux选 Bash)

### 第2步: 定义歌曲规格

```bash
musicify spec
```

或在 AI 编程助手中使用:
```bash
/spec
```

AI 会引导你填写:
- **歌曲类型**: 流行 (第一首歌推荐选择流行)
- **时长**: 3分30秒
- **风格**: 抒情 (适合新手)
- **语言**: 中文
- **受众**: 20-30岁, 中性
- **平台**: QQ音乐, 网易云

### 第3步: 主题构思

```bash
/theme
```

AI 会问你:
1. 想表达什么主题? (比如: 成长、爱情、梦想)
2. 为什么想写这个主题?
3. 希望听众听完后有什么感受?

**示例回答**:
> 主题: 追梦  
> 触发点: 想鼓励那些为梦想努力的人  
> 期待效果: 让听众感受到力量和希望

### 第4步: 情绪定位

```bash
/mood
```

根据你的主题,选择情绪氛围:
- 情绪色彩: 明亮
- 能量级别: 中等
- 节奏: 中速
- 氛围: 励志、温暖

### 第5步: 结构设计

```bash
/structure
```

AI 会为你创建标准流行歌曲结构:
```
Intro → Verse 1 → Pre-Chorus → Chorus → 
Verse 2 → Chorus → Bridge → Chorus → Outro
```

第一首歌建议使用标准结构。

### 第6步: 创作歌词

#### 选择创作模式:

**教练模式** (推荐新手学习):
```bash
/lyrics --mode coach
```
AI 会逐段引导你创作,提问式激发,不替你写词。

**快速模式** (快速原型):
```bash
/lyrics --mode express
```
AI 直接生成完整歌词,你可以在此基础上修改。

**混合模式** (平衡效率与原创):
```bash
/lyrics --mode hybrid
```
AI 生成框架,你填充细节。

**新手推荐顺序**:
1. 第一次: 用 `express` 模式看看 AI 怎么写
2. 第二次: 用 `hybrid` 模式,学习结构
3. 第三次: 用 `coach` 模式,完全原创

### 第7步: 押韵检查

```bash
/rhyme
```

AI 会检查:
- 押韵模式是否一致
- 韵母是否和谐
- 提供改进建议

### 第8步: 润色优化

```bash
/polish --focus all
```

AI 会优化:
- 用词精准度
- 意象丰富度
- 情感表达
- 可唱性

### 第9步: 导出歌词

```bash
/export --format txt
```

歌词会导出到 `export/` 目录。

## 🎯 推荐学习路径

### 阶段 1: 熟悉流程 (1-2首歌)

使用 **快速模式** 创作:
- 选择简单主题 (如: 友情、成长)
- 使用标准结构
- 观察 AI 如何写歌词
- 学习押韵和意象

### 阶段 2: 框架填充 (3-5首歌)

使用 **混合模式** 创作:
- 尝试不同主题
- 学习如何填充框架
- 理解段落功能
- 练习押韵

### 阶段 3: 原创创作 (6首歌+)

使用 **教练模式** 创作:
- 挑战复杂主题
- 完全原创
- 形成个人风格

## 💡 常见问题

### Q: 第一次用什么模式?
**A**: 推荐 `express` 模式,先看看 AI 怎么写。

### Q: 不知道写什么主题?
**A**: 从日常感受入手:
- 最近让你有感触的事
- 想对某人说的话
- 一个难忘的时刻

### Q: 押韵很难怎么办?
**A**: 使用 `/rhyme` 命令,AI 会提供押韵建议和替换词。

### Q: 生成的歌词不满意?
**A**: 
1. 检查 spec 和 theme 是否清晰
2. 使用 `/polish` 润色
3. 手动修改 `lyrics.md` 文件
4. 或重新运行 `/lyrics`

### Q: 如何提高原创度?
**A**: 使用教练模式,AI 只引导不代写。

## 🎨 创作技巧

### 1. 主题要具体

❌ 不好: "写一首关于爱情的歌"  
✅ 好: "写一首关于暗恋多年终于表白的歌"

### 2. 情感要真实

从自己的真实感受出发,不要编造。

### 3. 意象要具体

❌ 不好: "我很难过"  
✅ 好: "雨打湿了窗台,模糊了你的背影"

### 4. 押韵要自然

不要为了押韵而牺牲意义。

### 5. chorus 最重要

副歌要朗朗上口,最好能让人一听就记住。

## 🚀 进阶使用

### 尝试不同类型

- 🎸 摇滚: 力量感强,情绪直接
- 🎤 说唱: 节奏密集,多押韵
- 🎻 民谣: 故事性强,细腻真实
- 🏮 古风: 意象古典,韵味悠长

### 自定义结构

修改 `structure.json` 调整段落:
- 去掉 Intro (直接开始)
- 增加 Verse (更多叙事)
- 去掉 Bridge (保持简洁)

### 导出多种格式

```bash
/export --format txt  # 纯文本
/export --format md   # Markdown
/export --format pdf  # PDF
```
