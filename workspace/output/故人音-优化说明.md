# 《故人音》歌词格式优化说明

## 📊 优化对比

### 原始格式问题
```
[info]                          ❌ Suno/Tunee 不识别此标记
title: 故人音
artist: AI创作
reference: 探故知
key: A#
bpm: 120

[verse1]                        ⚠️ 小写，无空格（虽然能用，但不标准）
[chorus2]                       ⚠️ 同上
[chorus3]                       ⚠️ 重复副歌可以简化
```

### 优化后格式
```
[Verse 1]                       ✅ 标准格式：首字母大写 + 空格
[Chorus]                        ✅ 标准标记
[Bridge]                        ✅ 添加桥段增加变化
[Outro]                         ✅ 明确结束
[Fade Out]                      ✅ 淡出效果（可选）
```

### 2. **故人音-API格式.json**
**用途**: 用于 API 调用（Suno API, MusicAPI 等）

**特点**:
- ✅ JSON 格式，便于程序调用
- ✅ 元数据单独存储（title, style, tags, bpm, key）
- ✅ 歌词使用 \n 换行符
- ✅ 包含完整的风格描述

**使用示例**:
```javascript
// Suno API 调用示例
const response = await fetch('https://api.sunoapi.com/v1/sonic/create', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    custom_mode: true,
    mv: "sonic-v5",
    prompt: "国风, 忧伤惆怅, 古风流行",
    tags: "Chinese traditional, melancholic, poetic, guzheng, erhu",
    title: "故人音",
    lyrics: "[Verse 1]\n月沉江南 花影阑珊...",
    make_instrumental: false
  })
});
```

### 3. **故人音-Suno增强版.txt**
**用途**: Suno AI 专用增强版本

**特点**:
- ✅ 添加了 [Intro] 和 [Fade Out]
- ✅ 在标记中添加了乐器描述 `[Soft Instrumental Intro with Guzheng]`

**Suno 特殊技巧**:
- `[Intro]` 标记不太可靠，所以使用 `[Soft Instrumental Intro with Guzheng]`
- `[Fade Out]` 帮助 AI 更好地处理歌曲结尾

## 🎯 主要优化点

### 1. **段落标记标准化**
| 原格式 | 优化后 | 说明 |
|--------|--------|------|
| `[verse1]` | `[Verse 1]` | 首字母大写 + 空格 |
| `[chorus]` | `[Chorus]` | 标准格式 |
| `[chorus2]` | `[Chorus]` | 重复副歌简化 |
| `[chorus3]` | `[Chorus]` | 重复副歌简化 |

### 2. **元数据处理**
**原格式**:
```
[info]
title: 故人音
artist: AI创作
key: A#
bpm: 120
```

**优化方案**:
- 移除 `[info]` 标记
- 元数据通过 API 参数传递
- 或在 Suno 界面中单独填写

### 3. **结构优化**
**原结构**:
```
[verse1] → [chorus] → [verse2] → [chorus2] → [chorus3]
```

**优化后**:
```
[Intro] → [Verse 1] → [Chorus] → [Verse 2] → [Chorus] → [Bridge] → [Chorus] → [Outro]
```

**优势**:
- ✅ 添加了 [Bridge] 增加音乐变化
- ✅ 添加了 [Intro] 和 [Outro] 使结构完整
- ✅ 重复副歌简化，AI 会自动处理

## 🎵 平台使用指南

### Suno AI
1. **上传方式**:
   - 使用 "Custom Mode"
   - 将 `故人音-Suno增强版.txt` 的内容粘贴到歌词框
   - Style 填写: `国风, 忧伤惆怅, 古风流行`

2. **推荐设置**:
   - Model: Sonic v5
   - Instrumental: Off
   - BPM: 120
   - Key: A#

### Tunee AI
1. **上传方式**:
   - 使用 `故人音-优化版.txt`
   - 在风格描述中填写: `Chinese traditional, melancholic, poetic`

### MusicAPI / Suno API
1. **使用 JSON 格式**:
   - 直接使用 `故人音-API格式.json`
   - 或参考其中的格式编写 API 调用代码
