# /mood - 情绪氛围定位系统 🎭

> **核心理念**: 在主题基础上,进一步细化歌曲的情绪氛围
> **目标**: 完成 mood.json 的填写,明确情绪色彩、能量级别、节奏感觉

---

## 运行脚本 ⚠️ 必须执行

```bash
bash scripts/bash/mood.sh
```

AI根据返回的 spec 和 theme,引导用户定位情绪氛围。

---

## 引导流程

### 1. 情绪色彩

```
🎨 情绪色彩

基于你的主题 [theme],让我们确定歌曲的情绪色彩:

  1. 🌞 温暖 - 温暖人心,充满爱意
  2. ❄️ 冷峻 - 冷静客观,疏离感
  3. ✨ 明亮 - 积极向上,充满希望
  4. 🌑 灰暗 - 低沉压抑,阴郁氛围
  5. 🌈 缤纷 - 色彩斑斓,情绪多样
  6. 🎭 其他 (自己描述)

请选择 (1-6):
```

### 2. 能量级别

```
⚡ 能量级别

这首歌的能量强度如何?

  1. 🔥 高能 - 爆发力强,情绪激烈
  2. 💫 中等 - 平衡舒适,张弛有度
  3. 🌊 低能 - 平静舒缓,内敛深沉

请选择 (1-3):
```

### 3. 节奏感觉

```
🎵 节奏感觉

希望歌曲的节奏是:

  1. 🏃 快节奏 - 节奏明快,律动性强
  2. 🚶 中速 - 不快不慢,舒适自然
  3. 🧘 慢板 - 节奏舒缓,娓娓道来

请选择 (1-3):
```

### 4. 氛围标签

```
🏷️ 氛围标签

选择最符合这首歌的氛围标签 (可多选):

  1. 💕 浪漫 - 充满爱意和温情
  2. 😔 孤独 - 孤寂落寞
  3. 🦅 自由 - 无拘无束,释放自我
  4. 😞 压抑 - 沉重压抑
  5. 🎈 释放 - 宣泄释放
  6. 🌸 治愈 - 温暖疗愈
  7. 💪 力量 - 充满力量感
  8. 🌙 梦幻 - 缥缈梦幻
  9. 😊 欢快 - 轻松愉悦
  10. 😢 悲伤 - 感伤忧郁

请输入编号 (可多选,用逗号分隔,如: "1,6,8"):
```

---

## 保存配置

将用户选择保存到 mood.json:

```json
{
  "emotion_color": "温暖",
  "energy_level": "中等",
  "tempo": "中速",
  "atmosphere_tags": ["浪漫", "治愈", "梦幻"],
  "created_at": "[时间戳]",
  "updated_at": "[时间戳]"
}
```

展示结果并建议下一步。
