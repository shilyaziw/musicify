# /fill - 框架填充引导系统 ✏️

> **核心理念**: 逐项引导用户填充混合模式生成的框架
> **目标**: 完成所有 [待填充] 标记的内容

---

## 运行脚本 ⚠️ 必须执行

```bash
bash scripts/bash/fill.sh
```

脚本会扫描 lyrics.md,找出所有 [待填充] 标记。

---

## 填充流程

AI逐项引导填充:

```
╔══════════════════════════════════════════════════════════╗
║             填充项 1/12                                     ║
╚══════════════════════════════════════════════════════════╝

📍 位置: Verse 1 - 第2句
📝 要求: 描述具体场景或情境
🎯 押韵: 第4句需要与此句押韵

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 提示:
  • 要具体,不要抽象
  • 使用意象,让听众"看见"画面
  • 为后面的情感铺垫

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

请写出这句歌词:
```

每填充一项,进行检查并引导修改(如需要)。
