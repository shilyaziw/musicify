# /rhyme - 押韵检查系统 🎵

> **核心理念**: 分析歌词的押韵模式和质量,提供优化建议
> **目标**: 确保押韵清晰、和谐,提升歌词可唱性

---

## 运行脚本 ⚠️ 必须执行

```bash
bash scripts/bash/rhyme.sh
```

---

## 检查内容

AI分析每个段落的:
1. 押韵模式 (AABB/ABAB/ABCB等)
2. 韵母匹配度
3. 押韵词的质量
4. 提供改进建议
