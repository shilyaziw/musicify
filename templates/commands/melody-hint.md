# /melody-hint - 旋律提示系统 🎼

> **核心理念**: 为作曲提供参考,包括音域、节奏、旋律走向等
> **目标**: 帮助作曲人理解歌词的音乐需求

---

## 运行脚本 ⚠️ 必须执行

```bash
bash scripts/bash/melody-hint.sh
```

---

## 提示内容

AI根据歌曲类型、风格和情绪生成:

1. **音域范围**: 建议的人声音域 (如 C3-G4)
2. **建议BPM**: 节奏速度 (如 90-100 BPM)
3. **节奏型**: 建议的节奏型描述
4. **旋律走向**: 各段落的旋律方向(上扬/下沉/平稳)
5. **重音位置**: 哪些词应该加重
6. **参考歌曲**: 风格相似的歌曲推荐
