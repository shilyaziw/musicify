# /polish - 歌词润色系统 ✨

> **核心理念**: 多维度优化歌词质量
> **目标**: 提升用词、意象、情感表达和可唱性

---

## 运行脚本 ⚠️ 必须执行

```bash
bash scripts/bash/polish.sh --focus all
```

---

## 润色维度

### --focus wording (用词精准度)
- 检查用词是否精准
- 替换陈词滥调
- 避免重复

### --focus imagery (意象丰富度)
- 增强意象的具体性
- 使用更生动的比喻
- 丰富画面感

### --focus emotion (情感表达)
- 情感递进是否合理
- 情感强度是否到位
- 情感转折是否自然

### --focus singability (可唱性)
- 检查拗口的地方
- 调整字数分布
- 优化语气词

### --focus all (全面润色)
- 综合以上所有维度
